# Maze Chalenge

Algorithm created for evaluative activity of the data structure discipline

## Proposed
* run a .py script taking two arguments (input txt file and output file)
* read a .txt file with maze configuration
* transform maze to list item
* find a better way to get out of maze
* collect all artifacts in the way
* generate a output file with all informations about Teseu's way

## Python modules used
* [Sys](https://docs.python.org/3/library/sys.html) to read script arguments and finish script execution

## Requirements
* Python interpreter
